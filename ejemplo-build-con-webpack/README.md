# Vue 3 boilerplate

Vue 3 boilerplate

## Requirements
- Node 18.3.0

## Setup
Install npm dependencies
```
npm install
```

## Build assets
```
npm run build
```

## Start the server
```
npm run start
```

-------------------

## Dev environment
Webpack listener:
```
npm run watch
```

Server in dev mode:
```
npm run dev
```
